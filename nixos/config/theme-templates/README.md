# Theme Templates

This directory contains Jinja2 templates used by `generate-app-themes.py` to render
application configurations at runtime.

Expected templates (from the original project):
- dunstrc.j2
- rofi.rasi.j2
- yazi.toml.j2
- gtk2.j2
- gtk3.ini.j2
- gtk.css.j2 / gtk-dark.css.j2
- qt5ct-colors.conf.j2 / qt5ct-colors-dark.conf.j2
- qt5ct.conf.j2
- qt6ct.conf.j2
- xsettingsd.conf.j2
- kitty.conf.j2
- fcitx5.conf.j2 / fcitx5-dark.conf.j2
- nvim.lua.j2

Copy these from your existing `~/.config/theme-templates/`.
